# Notes API

A REST API built using Node.js and Express.

## Features

- Get all notes
- Search notes
- Get note by ID
- Create note
- Update note
- Delete note

## Tech Stack

- Node.js
- Express.js

## API Endpoints

GET /notes

GET /notes/:id

GET /notes?search=keyword

POST /notes

PUT /notes/:id

DELETE /notes/:id

## Run Locally

npm install

npm run dev
